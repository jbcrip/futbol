# Componentes

Esta carpeta contiene los componentes reutilizables de la aplicación.

## Estructura recomendada

Cada componente debe tener su propia carpeta con:
- `ComponentName.jsx` - El componente
- `ComponentName.css` - Estilos del componente (opcional)
- `index.js` - Export del componente

Ejemplo:
```
components/
└── Button/
    ├── Button.jsx
    ├── Button.css
    └── index.js
```
